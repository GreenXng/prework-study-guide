HTML

  • The head element contains information about the webpage.
  • Head element has information the user does not see.
  • The body element represents the visible content shown to the user.
  • The body can be broken up into more digestible snips of information using sections.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS file created will not work unless you link it to the HTML file(in the "head").
  • A class type can be added to specific "sections" and then the CSS rule can apply to all sections in that calss.

Git

  • Make sure you are in the correct folder before all else- cd bootcamp, cd prework-study-guide.
  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git pull origin main: this ensures that you are using in/pulling from the most up to date version of the main branch.
  • git add -A: adds the changes we made to our working directory to the statging area of Git.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
